## Problem

Merge k sorted Integer arrays and return it as one sorted array. Analyze and describe its complexity.

## Example:

Input:

    [
    [1, 4, 5],
    [1, 3, 4],
    [2, 6]
    ]

Output: `[1, 1, 2, 3, 4, 4, 5, 6]`
